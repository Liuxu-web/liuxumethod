## 懒人包,通用 JavaScript 方法.(不定期更新)

## 下载

自行选择一种

```javascript
npm i liuxumethod -S

cnpm i liuxumethod -S
```

## 使用

```JavaScript
import { isEmptyObject } from "liuxumethod";

const obj = {a:1};
console.log(isEmptyObject(obj)); // false
```

### 现有方法

```JavaScript
isEmptyObject, // 判断是否是空数组

isIdCard, // 验证身份证

isPhoneCard, // 验证手机号格式

getTime, // 获取时间

getWeek, // 获取星期

randomNum, // 随机数

randomColor, // 随机颜色

debouncing, // 防抖

throttle, // 节流

bubbleSort, // 冒泡排序

deWeight, // 数组去重

supplement, // 字符串前面补0
```
